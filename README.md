# PHP Event

User activity recorder.

## Requirement

- PHP 5.3 and highier

## Installation

Using composer:

`$ composer require tymfrontiers-cdn/php-event`

## Usage

```
use TymFrontiers;

```
Methods are self-explanatory, kindly check [Class file](src/Event.php).
